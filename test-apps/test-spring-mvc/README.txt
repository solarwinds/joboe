To test on the Spring webapp, use URLs as below:

http://localhost:8080/test-spring-mvc/

Simple Controller:
http://localhost:8080/test-spring-mvc/simple

Multi Controller:
http://localhost:8080/test-spring-mvc/multi/add
http://localhost:8080/test-spring-mvc/multi/delete
http://localhost:8080/test-spring-mvc/multi/list
http://localhost:8080/test-spring-mvc/multi/update

Annotated Controller:
http://localhost:8080/test-spring-mvc/annotated/annotated1
http://localhost:8080/test-spring-mvc/annotated/annotated2


Take note the Multi Controller and Simple Controller print testing exception stack trace to the console and those traces are expected
